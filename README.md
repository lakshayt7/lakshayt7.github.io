# Lakshay Tyagi GitHub Pages Portfolio

This folder contains a static GitHub Pages-ready portfolio site.

## Files

- `index.html`: site structure and content
- `styles.css`: custom styling and responsive layout
- `script.js`: small scroll-reveal animation
- `assets/`: profile image, project artwork, favicon, and resume

## Deploy on GitHub Pages

1. Create a repository named `lakshayt7.github.io` on GitHub.
2. Copy the contents of this folder into that repository.
3. Push to the `main` branch.
4. GitHub Pages will serve the site automatically from the repository root.

If you want to use a custom domain later, add a `CNAME` file to the repo root.
